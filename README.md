# 武汉理工社团管理系统

一个功能完善的高校社团管理系统，支持社团管理、活动组织、招新管理、财务管理等功能。

## 快速开始

1. 解压文件到任意目录
2. 运行oneclick_deploy.py
3. 浏览器打开 http://127.0.0.1:5000
4. 添加一个用户
5. 运行set_admin.py设置管理员即可开始使用
6. 再次运行运行oneclick_deploy.py即可

## 主要功能

### 🏫 社团管理

- 社团创建与审批
- 成员管理
- 社团信息编辑

### 📢 招新管理

- 发布招新公告
- 设置报名截止时间
- 在线报名和审核
- 自动添加成员

### 🎉 活动管理

- 活动发布与审批
- 活动报名（支持容量限制）
- 报名截止时间管理
- 活动评价系统

### 📦 物资管理

- 物资申请
- 物资审核

### 💰 财务管理

- 经费申请与审批
- 支出记录
- 财务报表（可按日期筛选）
- 可用额度实时更新

### 📨 消息系统

- 社团消息发布
- 重要消息标记
- 消息浏览和管理

### ⭐ 评价系统

- 活动评价（1-5星）
- 社团评价（成员专属）

## 技术栈

- **后端：** Python 3 + Flask
- **数据库：** SQLite
- **前端：** HTML + CSS + Jinja2 模板
- **安全：** Werkzeug 密码加密

## 注意事项

⚠️ **重要提示：**

- 本系统使用 Flask 开发服务器，仅适合开发测试环境
- 生产环境请使用 Gunicorn + Nginx 部署
- 请定期备份 `instance/clubmgr.db` 数据库文件
- 首次部署请修改 `SECRET_KEY`（在 app.py 中）

## 常见问题

### 端口被占用？

修改 `app.py` 最后一行的端口号：

```python
app.run(host="127.0.0.1", port=8080, debug=True)
```

### 找不到模块？

```bash
pip install -r requirements.txt
```

### 数据库错误？

删除 `instance/clubmgr.db`，重新运行程序会自动创建新数据库。
